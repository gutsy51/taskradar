# Task Radar

Прикладной проект в рамках ВКР для поиска проектов и заявок с платформ дистанционного найма.
Система упрощает доступ к разрозненным источникам данных: нормализует, индексирует и ранжирует данные.

## Стек

- `Django + DRF` - ORM, API, административный контур, миграции;
- `PostgreSQL + pgvector` - СУБД и расширение для хранения векторов;
- `Celery + RabbitMQ` - обработчик задач и менеджер сообщений;
- `Sentence Transformers` - построение семантических эмбеддингов;
- `Docker, nginx, GH Actions` - devops;

## Структура репозитория

```text
docs/        документация, аналитика, материалы ВКР
frontend/    клиентское приложение
ml/          поисковое ядро и эмбеддеры
producer/    сбор данных и интеграции с источниками
api/         Django + DRF backend
infra/       docker-compose, nginx, deployment-конфиги
```
