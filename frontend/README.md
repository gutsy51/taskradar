# PingTower

## Основные возможности

### Мониторинг
- **HTTP/HTTPS мониторинг** — проверка доступности веб-сайтов и API
- **ICMP пинг мониторинг** — проверка сетевой доступности серверов  
- **Настраиваемые интервалы** проверки от 30 секунд
- **Мультипроектная архитектура** для организации мониторов по группам

### Аналитика и отчеты
- **Метрики производительности** — время отклика, uptime, доступность
- **Графики и диаграммы** в реальном времени с использованием Recharts
- **Статистика по периодам** — от часа до месяца
- **Сводные отчеты** по проектам и командам

### Управление инцидентами  
- **Автоматическое обнаружение** проблем и сбоев
- **Трекинг активных инцидентов** с временными метками
- **История инцидентов** для анализа и улучшения
- **Уведомления** о критических изменениях статуса

### TV Mode — Режим отображения на больших экранах

**Специальный режим для демонстрации статуса на мониторах и TV в офисе:**

- **Полноэкранный интерфейс** — оптимизирован для просмотра с расстояния
- **Автообновление** — актуальные данные без вмешательства пользователя
- **Крупные индикаторы** — статусы видны даже с большого расстояния
- **Цветовая индикация** — мгновенная оценка состояния всех сервисов

TV Mode автоматически скрывает элементы управления и навигацию, оставляя только мониторы с их текущим статусом, что делает его идеальным для:
- Дисплеев в операционных центрах
- Мониторов в офисах разработки  
- Презентаций статуса сервисов на совещаниях

#### Как включить TV Mode

TV Mode активируется простым добавлением параметра `tv=true` к URL любой страницы панели:

```
# Обычный режим
http://localhost:3000/panel/1

# TV Mode
http://localhost:3000/panel/1?tv=true
```

#### Настройка TV Mode

1. **Откройте нужную панель проекта** — перейдите на страницу конкретного проекта
2. **Добавьте параметр tv=true** — в адресной строке добавьте `?tv=true` в конец URL
3. **Перейдите в полноэкранный режим** — нажмите F11 (или Cmd+Ctrl+F на Mac) для максимального эффекта. Также параметром `tab=incidents` можно настроить выбранную вкладку
4. **Настройте автообновление** — данные будут автоматически обновляться каждые 30 секунд

#### Особенности TV Mode

- **Скрытие интерфейса**: боковая панель, заголовок и элементы управления автоматически скрываются
- **Оптимизация для больших экранов**: увеличенные карточки мониторов с крупными индикаторами
- **Цветовая индикация**: 
  - Зеленый — сервис работает нормально
  - Желтый — есть предупреждения
  - Красный — сервис недоступен или есть ошибки
- **Автоматическое обновление**: данные обновляются без перезагрузки страницы
- **Минималистичный дизайн**: только критически важная информация

## Быстрый старт

### Установка зависимостей
```bash
bun install
```

### Запуск в режиме разработки
```bash
bun dev
```
Приложение будет доступно по адресу `http://localhost:3000`

### Запуск в продакшене
```bash
bun start
```

### Линтинг и форматирование
```bash
# Проверка кода
bun lint

# Автоматическое исправление
bun lint:write
```

## Технологический стек

### Frontend
- **React 19** — современная библиотека для UI
- **TypeScript** — типизированный JavaScript
- **Tailwind CSS** — утилитарный CSS фреймворк
- **shadcn/ui** — высококачественные компоненты

### Состояние и данные
- **Valtio** — реактивное управление состоянием
- **TanStack Query** — управление серверным состоянием
- **React Hook Form** — работа с формами
- **Zod** — валидация схем данных

### Визуализация
- **Recharts** — графики и диаграммы
- **Lucide React** — иконки
- **Framer Motion** — анимации

### Утилиты
- **Axios** — HTTP клиент
- **Day.js** — работа с датами  
- **React Router (Wouter)** — маршрутизация
- **Sonner** — уведомления

### Инструменты разработки
- **Bun** — быстрый JavaScript runtime и пакетный менеджер
- **Biome** — линтер и форматтер
- **TypeScript** — статическая типизация

## Структура проекта

```
src/
├── components/          # Переиспользуемые компоненты
│   ├── ui/             # Базовые UI компоненты (shadcn/ui)
│   └── app-*.tsx       # Компоненты приложения
├── pages/              # Страницы приложения
│   ├── home.tsx        # Главная страница с общей статистикой
│   ├── panels.tsx      # Список всех панелей мониторинга
│   ├── panel.tsx       # Детальная страница проекта с TV Mode
│   └── users.tsx       # Управление пользователями
├── dialogs/            # Модальные окна
├── hooks/              # Пользовательские хуки
├── lib/                # Утилиты и типы
├── states/             # Глобальное состояние (Valtio)
└── styles/             # Стили CSS
```

- **Темная/светлая тема** — автоматическое переключение
- **Drag & Drop** — удобная сортировка элементов
- **Живые обновления** — данные обновляются автоматически
- **Интуитивная навигация** — быстрый доступ к нужной информации

## Настройка и конфигурация

Проект использует современные инструменты конфигурации:
- `biome.json` — настройки линтера и форматтера
- `tsconfig.json` — конфигурация TypeScript
- `tailwind.config.js` — настройки Tailwind CSS
- `components.json` — конфигурация shadcn/ui
